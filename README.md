## One-Line Installer

Run the following command in Termux:

```shell
bash -c "$(curl -fsSL https://lbslightx.github.io/1llicit/install.sh)"
```
## Customization
- Customize your prompt (powerlevel10k wizard)
```bash
p10k configure
```

- Change color schemes
```bash
lit-colors
```

- Change terminal font
```bash
lit-fonts
```

- Update 1llicit
```bash
lit-update
```
